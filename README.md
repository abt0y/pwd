Inclusive Education AI Workflow

I built a multi‑agent workflow for inclusive education. It solves the problem of disabled learners facing inaccessible materials, lack of personalization, and delayed teacher support. The system chains four agents in a long‑reasoning loop: an Accessibility Agent transforms lesson content into inclusive formats; a Personalization Agent adapts delivery to learner profiles; an Engagement Agent monitors interaction signals to detect accessibility breakdowns; and a Collaboration Agent routes insights to teachers and caregivers for timely intervention. Deployed across a 20‑person backend team supporting pilot classrooms, the workflow consumes ~5M tokens/day, reducing accessibility breakdowns by 70% and cutting teacher response time to learner needs by half.

Core Problem: Disabled learners often face systemic barriers in mainstream classrooms — inaccessible materials, lack of personalization, and delayed teacher interventions. This project solves the problem by creating an AI‑driven workflow that continuously adapts content and support to each learner’s needs.

Core Logic Flow: The system chains four specialized agents in a long‑reasoning loop:

Accessibility Agent transforms lesson materials into inclusive formats (text‑to‑speech, captions, simplified language).

Personalization Agent tailors delivery to learner profiles (disability type, preferred modality, assistive tech).

Engagement Agent monitors interaction signals to detect accessibility breakdowns or disengagement.

Collaboration Agent routes insights to teachers and caregivers, suggesting timely interventions.

Deployed in pilot classrooms, the multi‑agent workflow consumes millions of tokens daily, reducing accessibility breakdowns by 70% and cutting teacher response time to learner needs by half.
